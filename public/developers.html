安装 SDK

pip install hjs-client

Node.js

npm install hjs-client
3

记录第一条事件

Python 示例：

from hjs_client import HJSClient

client = HJSClient(api_key="your-key")
result = client.record_judgment(
    entity="user@example.com",
    action="test"
)
print(result.id)
