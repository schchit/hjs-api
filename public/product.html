Record Judgment

POST /judgment

A Judgment binds:

  • ¬∑ actor identifier
  • ¬∑ decision content
  • ¬∑ contextual metadata
  • ¬∑ timestamp

A Judgment MUST include actor and timestamp.

View Example 
{
  "request": {
    "method": "POST",
    "url": "/judgment",
    "headers": {
      "Content-Type": "application/json"
    },
    "body": {
      "actor": "0x7F3a...e4b2",
      "decision": "approve_loan_application",
      "metadata": {
        "application_id": "LN-2026-02-22-001",
        "amount": 50000,
        "risk_score": 0.23,
        "model_version": "credit-scorer-v3"
      },
      "timestamp": "2026-02-22T14:35:00Z"
    }
  },
  "response": {
    "status": 201,
    "body": {
      "judgment_id": "jgd_9a7f5e3c1b2d",
      "status": "recorded",
      "proof_hash": "0x3f2a...c8e7",
      "timestamp": "2026-02-22T14:35:01Z"
    }
  }
}

Bind Delegation

POST /delegation

Delegation defines authority transfer.

A Delegation MUST reference a prior Judgment.

Delegation MAY define scope and expiry.

View Example 
{
  "request": {
    "method": "POST",
    "url": "/delegation",
    "headers": {
      "Content-Type": "application/json"
    },
    "body": {
      "judgment_id": "jgd_9a7f5e3c1b2d",
      "delegate_to": "0xB2c8...a93f",
      "scope": ["decisions.loan_under_100k", "data.view_credit_score"],
      "expires_at": "2026-03-22T23:59:59Z"
    }
  },
  "response": {
    "status": 201,
    "body": {
      "delegation_id": "dlg_4d8e2a1f6c3b",
      "status": "active",
      "expires_at": "2026-03-22T23:59:59Z"
    }
  }
}

Terminate Responsibility

POST /termination

Termination defines the end of responsibility.

Termination MUST reference a valid Judgment or Delegation.

View Example 
{
  "request": {
    "method": "POST",
    "url": "/termination",
    "headers": {
      "Content-Type": "application/json"
    },
    "body": {
      "reference_id": "dlg_4d8e2a1f6c3b",
      "reason": "project_completed",
      "terminated_by": "0x7F3a...e4b2"
    }
  },
  "response": {
    "status": 200,
    "body": {
      "termination_id": "trm_2b7c9d4e1f8a",
      "status": "terminated",
      "reference_id": "dlg_4d8e2a1f6c3b",
      "timestamp": "2026-02-22T15:20:00Z"
    }
  }
}

Verify Record

GET /verify/{id}

Verification is deterministic.

Integrity proof MAY be anchored to a public network.

Verification does not require trust in this service.

View Example 
{
  "request": {
    "method": "GET",
    "url": "/verify/jgd_9a7f5e3c1b2d"
  },
  "response": {
    "status": 200,
    "body": {
      "id": "jgd_9a7f5e3c1b2d",
      "exists": true,
      "record": {
        "actor": "0x7F3a...e4b2",
        "decision": "approve_loan_application",
        "timestamp": "2026-02-22T14:35:00Z"
      },
      "integrity": {
        "verified": true,
        "proof": "0x3f2a...c8e7",
        "anchored_at": "2026-02-22T14:40:00Z"
      }
    }
  }
}
