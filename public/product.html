结构化事件记录 API

每一次 AI 决策，都留下可追溯、可验证的结构化证据。

记录内容

  • · entity — 谁做的决定
  • · action — 做了什么决定
  • · scope — 在什么上下文中
  • · timestamp — 什么时候做的

技术规格

  • ✓ 传输加密 TLS 1.3
  • ✓ 存储加密 AES-256
  • ✓ 访问控制 API Key + 审计日志
  • ✓ 数据保留可配置（默认7年）
