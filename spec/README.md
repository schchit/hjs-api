# HJS 协议规范

**当前版本**：v1.0.0  
**许可证**：[CC0 1.0 公共领域贡献](LICENSE)

本协议遵循 [语义化版本 2.0.0](VERSIONING.md)。  
任何实现只要符合本规范，在同一个主版本号下应能互操作。  
协议变更通过 [RFC 流程](rfcs/) 进行。

---

## 协议概述

HJS 定义了四个核心原语——**判断事件（Judgment）**、**委托事件（Delegation）**、**终止事件（Termination）**、**归因关系（Attribution）**——作为一个最小、可组合的层，用于在社会技术系统中记录和追溯结构化事件流。

详细定义请参阅 [protocol.md](protocol.md)。
